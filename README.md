# git-rudimentos-masterclass
Rudimentos de git para explicar en una Masterclass al efecto

## Qué es Git
GIT es un sistema de control de versiones.

Sirve para llevar un registro de modificaciones en ficheros: 

* QUÉ se ha hecho
* CUÁNDO se ha hecho
* QUIÉN lo ha hecho

## Qué NO es Git

* No es un sistema para tener copias de seguridad en la nube... pero funciona realmente bien como sistema de copias de seguridad en la nube.
